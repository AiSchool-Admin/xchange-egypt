# السيناريو 1: البيع المباشر الكامل

## 1.1 إضافة منتج (البائع: test1)

1. افتح المتصفح واذهب إلى **https://xchange.eg**
2. اضغط زر **تسجيل الدخول**
3. ادخل حقل البريد: test1@xchange.eg
4. ادخل حقل كلمة المرور: Test@1234
5. اضغط زر **دخول**
6. اضغط زر **+ إضافة إعلان**
7. تظهر شاشة "ماذا تريد أن تفعل؟"
8. اضغط **عندي** (شيء للبيع أو المزاد أو المقايضة)
9. اختر نوع العرض: **بيع مباشر**
10. اضغط **التالي**
11. ادخل حقل العنوان: iPhone 15 Pro Max 256GB
12. ادخل حقل الوصف: آيفون 15 برو ماكس جديد بالكرتونة وجميع الملحقات
13. اضغط **اختر الفئة**
14. اختر **إلكترونيات**
15. اختر **هواتف**
16. اختر **Apple**
17. ادخل حقل السعر: 75000
18. اختر الحالة: جديد
19. اضغط **+ إضافة صور**
20. اختر 3 صور
21. اضغط **رفع**
22. اختر المحافظة: القاهرة
23. اختر المدينة: مدينة نصر
24. اضغط تبويب **الشحن**
25. فعّل **الشحن متاح**
26. ادخل تكلفة الشحن: 50
27. اضغط **نشر الإعلان**
28. ✅ تظهر: تم نشر إعلانك بنجاح
