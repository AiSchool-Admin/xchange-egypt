# السيناريو 1: البيع المباشر الكامل

## 1.1 إضافة منتج (البائع: test1)

1. افتح المتصفح واذهب إلى **https://xchange.eg**
2. اضغط زر **تسجيل الدخول**
3. ادخل حقل البريد: test1@xchange.eg
4. ادخل حقل كلمة المرور: Test@1234
5. اضغط زر **دخول**
6. اضغط زر **+ إضافة إعلان**
7. اختر **بيع مباشر**
8. اضغط **التالي**
9. ادخل حقل العنوان: iPhone 15 Pro Max 256GB
10. ادخل حقل الوصف: آيفون 15 برو ماكس جديد بالكرتونة وجميع الملحقات
11. اضغط **اختر الفئة**
12. اختر **إلكترونيات**
13. اختر **هواتف**
14. اختر **Apple**
15. ادخل حقل السعر: 75000
16. اختر الحالة: جديد
17. اضغط **+ إضافة صور**
18. اختر 3 صور
19. اضغط **رفع**
20. اختر المحافظة: القاهرة
21. اختر المدينة: مدينة نصر
22. اضغط تبويب **الشحن**
23. فعّل **الشحن متاح**
24. ادخل تكلفة الشحن: 50
25. اضغط **نشر الإعلان**
26. ✅ تظهر: تم نشر إعلانك بنجاح
